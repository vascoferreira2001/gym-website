# Gym Website

Projeto de website para ginásio desenvolvido com HTML, CSS, JavaScript, PHP e Bootstrap.

## Estrutura Inicial
- index.html
- assets/
  - css/
  - js/
  - images/
- pages/
- includes/

## Como usar
1. Edite os arquivos HTML, CSS, JS e PHP conforme necessário.
2. Utilize o Bootstrap para o layout e componentes visuais.
3. Coloque imagens e outros recursos na pasta assets.

## Requisitos
- PHP instalado para páginas dinâmicas.
- Navegador moderno para visualização.
